# AI Code Assistant

Web application for AI-assisted coding help — built with React, TypeScript, and Vite.

## Features

- Interactive AI chat for code questions
- Modern React UI with fast Vite dev server
- TypeScript throughout

## Getting started

```bash
git clone https://github.com/justsubway/ai-code-assistant.git
cd ai-code-assistant
npm install
npm run dev
```

Configure any required API keys in your environment (see `.env.example` if present).

## Tech stack

- React 18 · TypeScript · Vite · Tailwind CSS (if used in project)
